Table 4.5 (modified): Irreducible representations of line groups \(T_Q(f) D_n\) — family 5

The table below reproduces the structure of Damnjanović & Milošević, Line Groups in Physics, Table 4.5, but uses the helical-quantum-number convention adopted by the modified family-5 block of Irreps_tables_withparities.py. The factor group is generated by the screw \((C_Q\,|\,f)\), the axial proper rotation \(C_n\) (which equals \((C_Q\,|\,f)^{Q/n}\) modulo a lattice translation), and the perpendicular \(U\)-axis (\(C_2'\) at azimuth \(\alpha_U\)). Every other group element is a monomial in these generators. We tabulate the matrices of the three generators; the right-most column is the corresponding symmetry-adapted basis. We use \(n\) for the axial rotation order — for a \(D_n\) point group LineGroupAnalyzer returns \(\text{nrot} = 2n\), so the factor of two appearing in \(C_n\) below is built in.

Reality (in the sense of Damnjanović & Milošević, Sec. 4.3): the \({}_{0}A^{\Pi_U}_{m}\) rows are of the first kind (real). Each \({}_{k}\widetilde{E}^{(\cdot)}_{m}\) row at fixed \(k\neq 0\) is paired with its time-reversal partner at \(-k\).

What changed relative to the original tabulation

1. Axial-rotation phase. The original code used \({}_{C_n}\) phase \(=e^{i\,m\,2\pi/\text{nrot}}\). For a \(D_n\) point group LineGroupAnalyzer returns \(\text{nrot}=2n\), so this gives \(e^{i\,m\,\pi/n}\) — half of the correct value. The modified table uses the angle of the actual \(C_n\) generator (\(2\pi/n=4\pi/\text{nrot}\)), which is what is written above as \(e^{i\,m\,2\pi/n}\). On the test structure (\(n=2\), \(Q=6\)) this turns the Γ-point character overlap from \(0.72\) down to \(1.3\times 10^{-16}\).

2. m-range at Γ. At \(k=0\) the \((+m,-m)\) pair is already encoded inside the \({}_{0}E_{m}\) 2D matrix, so the loop must restrict to \(m\in\{0,1,\dots,\lfloor Q/2 \rfloor\}\). Without this restriction the original table over-counted with \(\sum d^2 = 20\) instead of \(|G|=12\).

3. \(m = Q/2\) splits off at Γ. For even \(Q\) the special value \(m\equiv-m \pmod{Q}\), so the 2D \({}_{0}E_{m}\) pairing collapses and the row must move to the 1D \({}_{0}A^{\Pi_U}_{m}\) branch with \(\Pi_U=\pm 1\). The current implementation flags this as is_special_m = (m==0) or (Q%2==0 and m==Q/2).

4. \((+k,-k)\) doubling at non-zero \(k\). The Bloch projector at non-zero \(k\) is built from the \(k\)-preserving little group \(L_k=T_Q\); the perpendicular \(U\) sits outside \(L_k\) because it sends \(+k\to -k\). Family 5 has no horizontal mirror, so a family-13-style 4D representation on \((|+m,+k\rangle, |-m,+k\rangle, |+m,-k\rangle, |-m,-k\rangle)\) would be reducible under the lone \(U\); the projector built from it returns \(\|P^2-P\|/\|P\|=1\). Two 2D blocks survive:

  • self-paired \(m\) (\(m=0\) or \(m=Q/2\)): basis \((|+k,m\rangle,|-k,m\rangle)\), the screw matrix carries opposite Bloch phases on the two \(k\)-sectors;
  • generic \(m\): basis \((|+k,+m\rangle,|-k,-m\rangle)\), induced from the 1D \(L_k\)-character \(\chi_{+m,+k}\).

Both blocks are themselves diagonal under the screw and the axial rotation, so the little-group filter inside get_adapted_matrix_withparities selects the right \(+k\)-sector character automatically. Enumerating all \(m\in\{0,1,\dots,Q-1\}\) gives one column-block of the adapted basis per \(L_k\)-character; the resulting six 2-mode blocks saturate \(3N=36\) at the test structure.
